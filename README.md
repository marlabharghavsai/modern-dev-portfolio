# Bharghav Sai - Personal Portfolio

A modern, animated, and fully responsive personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

[Insert Live URL Here]

## 🛠️ Technology Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Parallax:** react-scroll-parallax
- **Icons:** react-icons, lucide-react

## ✨ Features
- Fully responsive layout (Mobile, Tablet, Desktop)
- Parallax scrolling effects
- Dynamic on-scroll reveal animations
- Accessible design respecting `prefers-reduced-motion`
- Fast loading and optimized performance

## 📦 Local Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
